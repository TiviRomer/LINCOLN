# Logo Images

Place your LINCOLN logo image here.

## Supported Formats
- PNG (recommended)
- SVG
- JPG/JPEG
- WebP

## File Name
Name your logo file: `lincoln-logo.png` (or `.svg`, `.jpg`, `.webp`)

## Instructions
1. Download the logo image from the Grok Imagine link
2. Save it as `lincoln-logo.png` in this directory
3. The Logo component will automatically use it

If the image file doesn't exist, the component will fall back to the SVG logo.
